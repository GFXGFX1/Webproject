Описание проекта:
Веб-приложение для управления новостями - "Карась"
Общая информация:
Этот проект представляет собой веб-приложение, разработанное с использованием Flask — легковесного веб-фреймворка для Python. Приложение позволяет пользователям регистрироваться, входить в систему и управлять своими новостями. Оно предоставляет интерфейс для добавления, редактирования и удаления новостей, а также включает API для получения списка новостей в формате JSON.
Основные функции
1.Регистрация пользователей:
   - Пользователи могут создавать новые учетные записи, предоставляя имя пользователя и пароль.
   - Валидация данных на этапе регистрации.
2.Аутентификация:
   - Пользователи могут входить в систему с помощью своего имени пользователя и пароля.
   - Реализована возможность выхода из системы.
3.Дашборд:
   - После успешного входа пользователи перенаправляются на панель управления, где они могут видеть свои добавленные новости.
   - Каждая новость отображается с заголовком и содержимым.
4. Управление новостями:
   - Пользователи могут добавлять новые новости через специальную форму.
   - Может быть реализована функциональность редактирования и удаления новостей.
5.API:
   - Предоставляется RESTful API для получения списка новостей и отдельных новостей в формате JSON, что позволяет интегрировать приложение с другими сервисами.

#### Будущие улучшения:
- Добавить функциональность редактирования и удаления новостей.
- Реализовать системное уведомление о новых новостях.
- Добавить возможность комментирования и обсуждения новостей.
- Реализовать более сложные механизмы аутентификации (например, с использованием OAuth).
